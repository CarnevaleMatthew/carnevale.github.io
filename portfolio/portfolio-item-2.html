Project Two Our Journal

An Express App

This is my second project with Express/Node.js and MongoDB. This was created with my own code but it was based on a project that someone else has already done. I used this app to cement my knowledge of routes and middleware. All the things that I didn't understand well the first time I explored in more detail this time around. I included Google Auth(which was by far the hardest and most troublesome part of the creation of this app). Also I focused on cleaner code with comments and clearer file names and folders. It might not be extraordinary but I was proud that I was able to complete this with my own skill and through the use of docs from npm packages I didn't completely understand.

This app is called, 'Our Journal'. You can create, delete and update entries. You can register, login, logout. And by registering you can create entries, view public entries and create private entries. I wanted to make something that helped me understand what I had already learned about Express/Node.js and MongoDB(with Mongoose).
