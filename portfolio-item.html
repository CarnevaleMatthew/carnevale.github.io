Project Sanpo Trails

An Express App

This was my first Express project and it was a big one. This was a part of an Udemy course by Colt Steele, The Webdev Bootcamp 2021. The final project was a web app with full RESTful API functionality. I changed some of the focus to better suite my interests. Also this gave me a change to play around with the code and learn more from it. This taught me so much about Express/Node.js and MongoDB. This project is not perfect. But with each project I hope to improve. My goal is simple: Be better than I was yesterday. I hope to get a chance to show you I am always trying to improve and that I never give less than 100%.

This app is called, 'Sanpo Trails'. You can create, delete and update trails. You can register, login, logout. And by registering you can create trails, you can comment on trails and of course you can delete. The map, thank you mapbox, is included and points out the trails users have created. By clicking on a marker it will direct you to the show page so you can see the details on that trail.
